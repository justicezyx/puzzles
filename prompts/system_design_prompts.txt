# Prompts

Provide a high-level design to solve this system design interview problem:

```

Design a system to dispatch 1M images, each with 2MB in size, for human labelers to label cars in the image; there are 100 human labelers

```

Write your responses in succinct text, use bullet points.

With the following instructions:

1. First analyze the problem, clarify context, identify the key challenges, ask for any missing information might be important for solving this problem

2. Propose a high-level idea, make sure the high-level design cover the key challenges identified previous, reiterate the challenges and how the high-level idea addresses them

3. Confirm with the interviewer to get initial buy-in, revisit the design for any feedback from the interviewer

# Detailed design

Write your responses in succinct text, use bullet points.
Dive into the details to flesh out key design areas, identify Cloud systems and open source systems as candidate solutions

Wrap up and summarize the design, ask for any follow ups
